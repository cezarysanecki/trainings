Advent-of-Code
===================

**Projekt powstał w celu:**

- ukończenia wyzwania z Advent of Code 2020
- ukończenia wyzwania z Advent of Code 2021

W tym projekcie wykonano 25 zadań ze strony [Advent of Code 2020](https://adventofcode.com/2020).
W tym projekcie podjęto wyzwanie [Advent of Code 2021](https://adventofcode.com/2021).
